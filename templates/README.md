# Bgent Templates

Drop-in templates for bootstrapping your Bgent-managed project.

## Files

- [`AGENT_STATE.template.md`](AGENT_STATE.template.md) — Eisenhower kanban board template with 5 required sections (Deadlines / Constraints / Focus / Pending Actions / Achievements)
- [`meta_protocol.md`](meta_protocol.md) — Agent behavior constitution: initialization, closing ritual, memory management rules
- [`project_protocol.md`](project_protocol.md) — State-driven workflow template: session lifecycle, cross-conversation memory architecture
